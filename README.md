# kanban-board

A web platform designated for collaborative project management.

Created for the [Mobelite](https://www.mobelite.fr/) Summer 2025 internship.

## Technologies Used

- Next.js
- Shadcn
- Tanstack Query (React Query)
- MongoDB
- Auth0
- Socket.IO

## Running

- Clone the project
- Set up `.env` from `.env.example`
- `npm i`
- `npm run dev`
